# Abbreviation

Use the `<BreadCrumbs/>` `<BreadCrumbsItem/>` component for rendering breadcrumbs.

## Props
<h4>BreadCrumbs</h4>
<Props of={BreadCrumbs} />
<h4>BreadCrumbsItem</h4>
<Props of={BreadCrumbsItem} />

## Examples

<Preview>
  <BreadCrumbs>
      {["Step 1", "Step 2", "Step 3"].map(item => (
        <BreadCrumbsItem
          key={item}
          onClick={() => {}}
          active={"Step 1" === item}
        >
          {item}
        </BreadCrumbsItem>
      ))}
    </BreadCrumbs>
</Preview>

## Usage

```js
import React from 'react';
import { BreadCrumbs, BreadCrumbsItem } from 'src/components';

const MyPage = () => (
  <div>
<BreadCrumbs>
<BreadCrumbsItem onClick={()=>{}} active={true}> Step 1</BreadCrumbsItem>
<BreadCrumbsItem> Step 2</BreadCrumbsItem>
<BreadCrumbsItem> Step 3</BreadCrumbsItem>
</BreadCrumbs>
  </div>
);
