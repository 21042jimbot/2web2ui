# ExternalLink

Use the `<ExternalLink/>` component to render a link that directs to a new page outside of the app.

## Props

<Props of={ExternalLink} />

## Examples

### Default

<Preview>
  <ExternalLink to="https://developers.sparkpost.com">
    SparkPost Developer Documentation
  </ExternalLink>
</Preview>

### As Button

<Preview>
  <ExternalLink as={Button} to="https://developers.sparkpost.com" primary>
    SparkPost Developer Documentation
  </ExternalLink>
</Preview>

## Usage

```js
import React from 'react';
import { ExternalLink } from 'src/components';

const MyComponent = () => (
  <ExternalLink to="https://developers.sparkpost.com">
    SparkPost Developer Documentation
  </ExternalLink>
);
```
