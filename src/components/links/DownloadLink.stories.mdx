# DownloadLink

Use the `<DownloadLink/>` component when the user needs to download a file.

## Props

<Props of={DownloadLink} />

## Examples

### Default

<Preview>
  <DownloadLink to={exampleFilePath}>Download Me!</DownloadLink>
</Preview>

### As Button

<Preview>
  <DownloadLink as={Button} to={exampleFilePath} primary>
    Download Me!
  </DownloadLink>
</Preview>

### With Filename

<Preview>
  <DownloadLink as={Button} download="example.csv" to={exampleFilePath} primary>
    Download Me!
  </DownloadLink>
</Preview>

## Usage

```js
import React from 'react';
import exampleFilePath from './example-file.csv';
import { DownloadLink } from 'src/components/links';

const MyComponent = () => <DownloadLink to={exampleFilePath}>Download Me!</DownloadLink>;
```
