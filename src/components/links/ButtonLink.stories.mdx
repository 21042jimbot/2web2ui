# ButtonLink

Use the `<ButtonLink/>` when a semantic HTML `<button/>` is needed (i.e., the element has an `onClick` prop) but it should visually look like a link.

## Props

<Props of={ButtonLink} />

## Examples

### Default

<Preview>
  <ButtonLink onClick={() => console.log('onClick!')}>Click Here</ButtonLink>
</Preview>

## Usage

```js
import React from 'react';
import { ButtonLink } from 'src/components';

const MyComponent = () => (
  <ButtonLink onClick={() => console.log('onClick!')}>Click Here</ButtonLink>
);
```
