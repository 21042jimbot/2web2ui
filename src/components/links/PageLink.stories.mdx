# PageLink

The `<PageLink/>` is a simple wrapper for the `<UnstyledLink/>` component wired to react-router.

## Props

<Props of={PageLink} />

## Examples

### Default

<Preview>
  <MemoryRouter>
    <PageLink to="/dashboard">Dashboard</PageLink>
  </MemoryRouter>
</Preview>

### As Button

<Preview>
  <MemoryRouter>
    <PageLink as={Button} primary to="/dashboard">
      Dashboard
    </PageLink>
  </MemoryRouter>
</Preview>

## Usage

```js
import React from 'react';
import { PageLink } from 'src/components/links';

const MyComponent = () => <PageLink to="/dashboard">Dashboard</PageLink>;
```
