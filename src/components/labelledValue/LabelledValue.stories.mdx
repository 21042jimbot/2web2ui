# LabelledValue

The `<LabelledValue/>` component is used to style labels and their associated values in a side-by-side manner.

## Props

<Props of={LabelledValue}/>

## Examples

### Default

<Preview>
  <Story name="default">
    <div>
      <LabelledValue label="First Name" value="Sparky"/>
      <LabelledValue label="Last Name" value="SparkPost"/>
      <LabelledValue label="Phone Number" value="(443) 555 - 1234"/>
    </div>
  </Story>
</Preview>

## Usage

```js
import React from 'react';
import LabelledValue from 'src/components/labelledValue/LabelledValue';

const MyComponent = () => (
  <div>
    <LabelledValue label="First Name" value="Sparky"/>
    <LabelledValue label="Last Name" value="SparkPost"/>
    <LabelledValue label="Phone Number" value="(443) 555 - 1234"/>
  </div>
);
```
