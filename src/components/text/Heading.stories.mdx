# Heading

The `<Heading/>` component is used to render heading elements - these elements can be semantically
one heading level, but visually another.

## Props

<Props of={Heading} />

## Examples

<Preview>
  <div>
    <Heading as="h1">This is a level 1 heading</Heading>
    <Heading as="h2">This is a level 2 heading</Heading>
    <Heading as="h3">This is a level 3 heading</Heading>
    <Heading as="h4">This is a level 4 heading</Heading>
    <Heading as="h5">This is a level 5 heading</Heading>
    <Heading as="h6">This is a level 6 heading</Heading>
    <Heading as="h2" looksLike="h6">
      This is a level 2 heading, but it looks like a level 6 heading
    </Heading>
  </div>
</Preview>

## Usage

```js
import React from 'react';
import { Heading } from 'src/components/text';

const MyComponent = () => {
  return (
    <Heading as="h2" lookLike="h6">
      This is a level 2 heading, but it looks like a level 6 heading
    </Heading>
  );
};
```
