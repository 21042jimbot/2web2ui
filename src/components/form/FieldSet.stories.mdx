# FieldSet

Use the `<FieldSet/>` component to group related form fields.

## Props

<Props of={FieldSet} />

## Examples

### Default

<Preview>
  <FieldSet legend="Full Name">
    <label>
      First Name
      <input name="first" type="text"></input>
    </label>
    <label>
      Last Name
      <input name="last" type="text"></input>
    </label>
  </FieldSet>
</Preview>

## Usage

```js
import React from 'react';
import { FieldSet } from 'src/components/form';

const NameFields = () => (
  <FieldSet legend="Name">
    <label>
      First Name
      <input name="first" type="text" />
    </label>

    <label>
      Last Name
      <input name="last" type="text" />
    </label>
  </FieldSet>
);
```
