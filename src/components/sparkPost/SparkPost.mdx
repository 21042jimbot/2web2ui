# SparkPost

Use the `<SparkPost/>` component when a SparkPost logo is needed.

## Usage

```js
import React from 'react';
import SparkPost from 'src/components/sparkPost/SparkPost';
import CenteredLogo from 'src/components/sparkPost/CenteredLogo';

const MyComponent = () => (
  <>
    <SparkPost.Logo />
    <CenteredLogo />
    <SparkPost.Icon />
  </>
);
```

## Examples

### Logo

```js
<SparkPost.Logo />
```

### Centered Logo

```js
<CenteredLogo />
```
