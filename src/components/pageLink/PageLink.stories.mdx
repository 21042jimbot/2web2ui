# PageLink

The `<PageLink/>` is a simple wrapper for the `<UnstyledLink/>` component wired to react-router. 

## Props

<props of={PageLink}/>

## Examples

### Default

<Preview>
  <Router>
    <PageLink to="https://sparkpost.com">SparkPost.com</PageLink>
  </Router>
</Preview>

## Usage

```js
import React from 'react';
import PageLink from 'src/components/pageLink';

const MyComponent = () => <PageLink to="https://sparkpost.com">SparkPost.com</PageLink>;
```
