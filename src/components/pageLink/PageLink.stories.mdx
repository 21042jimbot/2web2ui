# PageLink

The `<PageLink/>` is a simple wrapper for the `<UnstyledLink/>` component. Using this component instead of `<UnstyledLink/>` allows app developers to manage a single point of entry for the Matchbox component.

## Props

<props of={PageLink}/>

## Usage

```js
import React from 'react';
import PageLink from 'src/components/pageLink';

const MyComponent = () => <PageLink to="https://sparkpost.com">SparkPost.com</PageLink>;
```
