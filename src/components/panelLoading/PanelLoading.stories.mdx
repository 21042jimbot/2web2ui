# Panel Loading

Use the `<PanelLoading/>` component to create a loading state within a `<Panel/>` container. This is particularly useful within `<Modal/>` components.

## Props

<Props of={PanelLoading}/>

## Usage

```js
import React from 'react';
import { Panel } from '@sparkpost/matchbox';
import { PanelLoading } from 'src/components';

const MyComponent = () => (
  <Panel>
    <PanelLoading accent/>
  </Panel>
)
```
