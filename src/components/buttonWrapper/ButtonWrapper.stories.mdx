# ButtonWrapper

Use the `<ButtonWrapper/>` to wrap Matchbox `<Button/>` components and provide consistent spacing, typically within forms.

## Examples

### Default

<Preview>
  <Story name="default">
    <Panel>
      <Panel.Section>
        <TextField
          id="first-name"
          name="firstName"
          label="First Name"
          type="text"
        />
        <TextField
          id="last-name"
          name="lastName"
          label="Last Name"
          type="text"
        />
        <ButtonWrapper>
          <Button primary>Save</Button>
        </ButtonWrapper>
      </Panel.Section>
    </Panel>
  </Story>
</Preview>

## Usage

```js
import React from 'react';
import { Panel, Button, TextField } from '@sparkpost/matchbox';
import { ButtonWrapper } from 'src/components';

const MyComponent = () => (
  <Panel>
    <Panel.Section>
      <TextField
        id="first-name"
        name="firstName"
        label="First Name"
        type="text"
      />
      <TextField
        id="last-name"
        name="lastName"
        label="Last Name"
        type="text"
      />
      <ButtonWrapper>
        <Button primary>Save</Button>
      </ButtonWrapper>
    </Panel.Section>
  </Panel>
)
```
