# SubduedText

Use `<SubduedText/>` to render subdued text for the Hibana version of the app. In the OG theme, the
text renders as a default paragraph.

## Props

<Props of={SubduedText} />

## Examples

<Preview>
  <AccentText>Hello, world</AccentText>
</Preview>

## Usage

```js
import React from 'react';
import { AccentText } from 'src/components';

const MyComponent = () => <AccentText>Hello, world</AccentText>;
```
