# ActionPopover

Use the `<ActionPopover/>` for simple `<Popover/>` dropdowns.

## Props

<Props of={ActionPopover}/>

## Examples

### Default

<Preview>
  <Story name="default">
    <ActionPopover actions={[{ content: 'Edit', to: '/edit' },{ content: 'Save', to: '/save' },{ content: 'Delete', to: '/delete'}]}/>
  </Story>
</Preview>

## Usage

```js
import React from 'react';
import ActionPopover from 'src/components/actionPopover';

const MyComponent = () => (
  <ActionPopover actions={[
    { content: 'Edit', to: '/edit' },
    { content: 'Save', to: '/save' },
    { content: 'Delete', to: '/delete'}
  ]}/>
);
```
