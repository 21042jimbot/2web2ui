# FullBanner

Use the `<FullBanner/>` to display a full width alert that spans its container.

## Props

<Props of={FullBanner} />

## Examples

### Default

<Preview>
  <FullBanner>
    <Text>Here is some content</Text>
  </FullBanner>
</Preview>

### With a Close Button

<Preview>
  <FullBanner>
    <Text>Here is some content</Text>
    <FullBanner.CloseButton onClick={() => console.log('onClick!')} />
  </FullBanner>
</Preview>

### With an Icon

<Preview>
  <FullBanner>
    <FullBanner.Icon as={AccessTime} />
    <Text>Here is some content</Text>
    <FullBanner.CloseButton onClick={() => console.log('onClick!')} />
  </FullBanner>
</Preview>

### With a Link

Note that this link semantically defaults to `ButtonLink` and assumes there is a passed in click handler. Use the `as` prop if a different type of link component is needed.

<Preview>
  <FullBanner>
    <Text>Here is some content</Text>
    <FullBanner.Link onClick={() => console.log('onClick!')}>Click Me</FullBanner.Link>
  </FullBanner>
</Preview>

### Usage

```js
import React from 'react';
import { FullBanner } from 'src/components/fullBanner';

const MyComponent = () => (
  <FullBanner>
    <FullBanner.Icon as={AccessTime} />
    <Text>Here is some content</Text>
    <FullBanner.CloseButton onClick={() => console.log('onClick!')} />
  </FullBanner>
);
```
